infographic-dot-clock
A grid-based, infographic time machine.

数字を介さず、グリッド上のドットの疎密と流動によって「時間の解像度」を可視化する実験的クロック。

🌑 Concept
このクロックは、時計としての実用性を超え、情報（Infographic）が身体的なリズム（Dot Motion）へと変換される過程を表現しています。

Background Layers: 月、日、曜日が背景を静かに、かつ一定の速度で横断し、大きな時間の流れ（暦）を形成します。

Core Mechanics: 画面中央では、秒針・分針・時針に相当するドットの塊が、グリッドの制約を受けながら有機的に変容し、現在の瞬間を刻みます。

Visual DNA: 1960年代のミニマル・アートと現代のデジタル・タイポグラフィの交差点を目指した、白と黒のみの世界観。

🤖 Collaborative Process
Sugawara (Vision): グリッドの海におけるドットの重なり、および「数字に頼らない時間」のインフォグラフィック定義。
Gemini (Implementation): Canvas APIによる高密度なドット・アニメーションと、時間軸による座標変換ロジックの構築。
